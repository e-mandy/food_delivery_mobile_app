# APPLICATION MOBILE DE LIVRAISON DE NOURRITURE

## DESCRIPTION
Une application mobile destinée à permettre à tout utilisateur, de passer une commande de nourriture.

## INTÉRÊTS
Ce projet s'inscrit dans une démarche académique et vise à renforcer les compétences et les aptitudes des membres de l'équipe en développement d'application Flutter.

### COMMENT ACCÉDER AU RENDU

  Nous allons suivre ces étapes : 

    git clone https://github.com/e-mandy/food_delivery_mobile_app.git

  Accédez aux dossiers ou fichier depuis Android Studio ou un autre éditeur de texte puis :

    flutter pub get

  Lorsque les dépendances seront installées, vous pourrez maintenant run le projet et accédez à la vue : 

    flutter run
